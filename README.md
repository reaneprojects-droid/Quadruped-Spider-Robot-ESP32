# Quadruped Spider Robot using ESP32

A four-legged spider-inspired robot using ESP32 and 12 servo motors with Bluetooth mobile control and inverse kinematics.

## Robot Prototype
![Spiderbot](images/spiderbot_front.jpg)

## Hardware Used
- ESP32 Microcontroller
- ESP32 Expansion Board
- 12 Micro Servo Motors
- 3D Printed Chassis
- 12V Battery
- Dual Buck Converters

## Features
- Quadruped locomotion
- Servo calibration system
- Bluetooth mobile control
- Mobile application built using MIT App Inventor

## Code
The project contains two main programs:
- `spiderbot_servo_calibration.ino` – centers all servos at 90° for horn alignment
- Main control code – handles gait motion and Bluetooth commands
